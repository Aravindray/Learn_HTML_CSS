Notes:

:is pseudo class

instead of use 2 different selectors like nav a:focus, nav a:hover we may use nav :is(a:focus, a:hover). This is also based on specificity.

:any-link pseudo class

Instead of using a:link, a:visited use a:any-link to keep you code DRY!

:where pseudo class

This is similar to :is but it will not consider specificity.

:target pseudo class

We may use it to highlight the target element by anchor tag.

Do you know how to select attribute of an element?

use the square bracket [], for example img[alt].

:not pseudo class

If you want to select the particular element with absence of specific attribute we may use :not pseudo class, for example img:not([not])

nth-child pseudo class

<section class="container">
<div> </div>
<div> </div>
<div> </div>
<div> </div>
<div> </div>
</section>
From the above code if we select the section:nth-child(2), it will not select the 2nd div element, it select the 2nd section element inside the body element or from it's ancestor element. So, if we want to select the second div element use this selector .container div:nth-child(2). Another values we may use odd, even
