Notes

Static - default
Relative - alloted space is not changing, but the content/element itself is moving.
Absolute - moved out of the document body, alloted space is changing, bottom of zero is not bottom of the document, but bottom of viewport. not use in regular basis.
Fixed - moved out of the document body, alloted space is changing
Sticky - combination of relative and fixed positions
